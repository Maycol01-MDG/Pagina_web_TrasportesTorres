# Pagina_web_TrasportesTorres

Web App moderna, rápida y "Mobile-First" para una empresa de transportes de pasajeros y encomiendas en Perú.

## Características
- 🚌 **Selector de Rutas**: Cajamarca - Tarapoto, Tarapoto - Chiclayo, Chota - Chiclayo.
- 📱 **Mobile-First**: Diseño premium optimizado para dispositivos móviles.
- 💬 **WhatsApp Dinámico**: Contacto directo con oficinas según la ruta seleccionada.
- 📦 **Gestión de Encomiendas**: Sección dedicada para cotización de envíos.
- 🎨 **UX/UI Moderna**: Animaciones suaves e iconos intuitivos.

## Tecnologías
- HTML5 Semántico
- CSS3 (Variables, Flexbox, Grid)
- JavaScript Vanilla
- Lucide Icons
